HTML Tag and Attribute Cheat Sheet

HTML Tags

Tag Name Description
<!--...--> Comment Tag Defines a comment
<!DOCTYPE> Document Type Tag Defines the document type
<a> Hyperlink Tag Defines a hyperlink
<article> Article Tag Defines an article
<aside> Aside Tag Defines content aside from page main content
<audio> Audio Tag Defines embedded sound content
<body> Body Tag Defines the document's main content
<br> Break Tag Defines a single line break
<button> Button Tag Defines a clickable button
<caption> Caption Tag Defines a table caption
<div> Div Tag Defines a section in a document
<em> Italics Tag Defines emphasized text in italics
<figcaption> Figure Caption Tag Defines a caption for a <figure> element
<figure> Figure Tag Specifies self-contained content
<footer> Footer Tag Defines a footer for a document or section
<form> Form Tag Defines an HTML form for user input
<h1> to <h6> Heading Tags Defines HTML headings with h1 being the largest and h6 being the smallest
<head> Head Tag Contains metadata/information for the document
<header> Header Tag Defines a header for a document or section
<html> HTML Tag Defines the root of an HTML document. All HTML code must be placed within an html tag
<img> Image Tag Defines an image. Is a self closing tag
<link> Link Tag Defines the relationship between a document and an external resource. Typically used to link style sheets
<main> Main Tag Specifies the main content of a document
<nav> Nav Tag Defines navigation links
<ol> Ordered List Tag Defines an ordered list i.e a numbered list
<p> Paragraph Tag Defines a paragraph
<section> Section Tag Defines a section in a document
<span> Span Tag Defines a section in a document. Typically used to separate a small section of text from a larger text
<strong> Strong Tag Defines important text and displays them in bold
<table> Table Tag Defines a table
<tbody> Table Body Tag Defines the main content of a table
<td> Table Data Tag Defines a cell in a table
<tfoot> Table Footer Tag Groups the footer content in a table
<th> Table Header Cell Tag Defines a header cell in a table. Must be inside a <tr> tag
<thead> Table Header Tag Groups the header content for a table
<time> Time Tag Defines a specific time
<title> Title Tag Defines a title for a document
<tr> Table Row Tag Defines a row in a table
<ul> Unordered List Tag Defines an unordered list. Is a bulleted list
<video> Video Tag Defines embedded video content

Attributes

Attribute Name Tag Association Description
autoplay Autoplay Attribute <audio>,<video> Specifies that the audio/video will start playing as soon as it is ready
class Class Attribute Any tag Specifies one or more classnames for an element (refers to a class in a style sheet)
colspan Column Span Attribute <td>,<th> Specifies the number of columns a table cell should span
controls Controls Attribute <audio>,<video> Specifies that audio/video controls should be displayed
dir Direction Attribute Any tag Specifies the text direction for the content in an element
href Reference Attribute <a>,<area>,<base>,<link> Specifies the URL of the page the link goes to
id ID Attribute Any Tag Specifies a unique id for an element
